
# GMOVIE

## (Global Meeting Online Video Interactive Experience)

---

**WebRTC를 활용한 화상회의 플랫폼**

GMOVIE는 실시간으로 회의를 진행하고 동시에 녹음하며, 자동으로 회의록을 작성하는 기능을 제공합니다. 더불어, 필요한 경우 회의록을 간략하게 요약하는 기능을 통해 사용자들은 불편함 없이 회의를 관리할 수 있게 됩니다.
